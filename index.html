Interactive Pie Chart

Description: This chart visualizes the proportion of different License Types by City. Users can filter by selecting a city from the dropdown to explore the data for that city.

Design Choices: The chart encodes the percentage of each License Type as a slice of the pie chart and colors each slice according to the License Type. I chose this design to make it easy to compare the proportions visually. Data transformations involved aggregating counts of each License Type and calculating their percentage.

Interactivity: I added a dropdown filter to allow users to select a city. This makes the chart interactive and helps users explore the data in more detail.

Bar Chart

Description: This bar chart visualizes the top 5 License Statuses in Chicago based on frequency. The chart shows the count of each License Status.

Design Choices: The `count` is mapped to the `y` axis and the `License Status` is mapped to the `x` axis. I used different colors for each License Status to enhance readability and comparison. Data was transformed by selecting the top 5 License Statuses and combining others into a single "Others" category.

Interactivity: This chart does not have additional interactivity but is still effective for comparing the top 5 License Statuses in Chicago.
