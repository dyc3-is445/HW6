First Visualization: Percentage Distribution of License Types Across Cities

Description: This chart visualizes the percentage distribution of different license types across various cities. Users can select a city from the dropdown to explore the data for that city.

Design Choices: In this pie chart, the `theta` encoding represents the proportion of each license type, while the `color` encoding is used to distinguish between license types. The slices of the pie chart are color-coded based on license type to make the visualization easy to interpret. Data transformations involved aggregating counts of each license type and calculating the percentage for each type.

Interactivity: A dropdown filter allows users to select a city, making the chart interactive. This interactivity enables users to explore license type distributions for individual cities, providing a more detailed view of the data.

Second Visualization: Bar Chart of License Status in Chicago

Description: This bar chart visualizes the top 5 most common license statuses in Chicago. The chart shows the count of each license status, with any additional categories combined into an "Others" group.

Design Choices: In this bar chart, the `x` axis represents the different license statuses, and the `y` axis shows the count for each status. Different colors are used for each license status to improve readability and facilitate comparison. The data was transformed to show only the top 5 most frequent statuses, with other statuses grouped into the "Others" category.

Interactivity: This chart does not have additional interactivity, but it is effective for visually comparing the top 5 license statuses in Chicago.
